# api_yatube

<h2>Запуск:</h2>
<pre>
<code>virtualenv venv</code>
</pre>
<pre>
<code>source venv/bin/activate</code>
</pre>
<pre>
<code>pip3 install -r requirements.txt</code>
</pre>
<pre>
<code>python3 manage.py runserver</code>
</pre>
